# 子供のきもち - Project Context

## 概要
「子供のきもち」は、子どもが怒り・不安・悲しみ・イライラなどの強い感情を感じた瞬間に使う感情サポートアプリです。  
子ども本人には「今できる行動」を示し、親には対応方法や声かけ例、振り返り機能を提供します。

このアプリは単なる感情記録アプリではありません。  
主目的は、感情が起きた瞬間の対応支援と、感情調整の成功体験を積み重ねることです。

## コンセプト
- Emotion Coaching（感情コーチング）がベース
- 子どもの感情を否定しない
- 感情を言語化しやすくする
- 落ち着くための具体行動を示す
- 小さな成功体験を明確に褒める
- 忙しい親でもすぐ使える操作性にする

## 想定ユーザー
- 3歳〜10歳前後の子どもを育てる親
- 実際には子ども本人と保護者の両方が使う

## コア体験
1. 感情を選ぶ
2. 感情詳細画面を開く
3. 子ども向けの共感メッセージとイラストを表示する
4. 「いまやること」を1つだけ提示する
5. 子どもが実行する
6. 「できた！」を押す
7. 褒め言葉と演出で成功体験を返す
8. 履歴に残し、必要に応じて親が振り返る

## UX方針
- 子ども向けUIを最優先にする
- 情報を詰め込みすぎない
- 抽象的な説明より具体行動を優先する
- 子どもに複数選択を強いすぎない
- 親向け情報は補助として扱う
- 記録や分析は補助機能として扱う
- 一部分析機能はプレミアム機能とする

## AIが理解すべき重要事項
- このアプリは「感情記録アプリ」ではなく「感情対応アプリ + 振り返り支援アプリ」
- 最優先は子ども向けUI
- 「今やること」は1つだけ提示する思想を崩さない
- 成功演出はアプリ価値の一部
- 親向け情報は最初から出しすぎない
- レポート機能は重くしすぎず、全体体験を悪化させない

## 一文要約
「子供のきもち」は、子どもの感情が高ぶった瞬間に、子どもには今できる行動を示し、親には対応のヒントと振り返り機能を提供する感情サポートアプリです。
